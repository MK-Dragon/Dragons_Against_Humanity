# Dragons_Against_Humanity
A fun D&amp;D take on the popular game "cards against humanity"! Draw some cards and create THE Most Fun and Chaotic story you can! (Best Played with FUN Drunk and Frieds)

<br><br><hr><hr><br><br>

# How to RUN:
```bash
cd paises-api
npm install
npm run dev
```
